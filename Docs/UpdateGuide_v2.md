# Inspector Manager アップデートガイド

今回のアップデートでは、Inspectorの更新ロジックを刷新し、安定性と視認性を大幅に向上させました。主な変更点は以下の通りです。

## ✨ 新機能

### 1. スマート更新 (Smart Rotation)
Inspectorの内容切り替え処理を全面的に書き換えました。
- **高速化**: 従来の一瞬アンロックする方式から、内部APIを利用した直接書き換え方式に変更しました（Unity 2022+対応）。
- **安定化**: 処理の競合を防ぎ、意図しない書き換わりやエラーを削減しました。

### 2. ビジュアルフィードバック
どのInspectorが更新されたか、次はどれが更新されるかが一目でわかるようになりました。

#### オーバーレイ (Inspector上部)
- **更新フラッシュ**: 内容が更新されたInspectorのロックボタンが一時的に**緑色**に光ります。
- **次候補バッジ**: 次に選択したオブジェクトが表示される予定のInspectorに **[▶ Next]** バッジが表示されます。

#### ステータスビュー (Statusタブ)
- **ハイライト**: 更新されたばかりのInspectorがリスト上でハイライト表示されます。
- **順序表示**: 次に更新されるInspectorに **[NEXT]** マークがつきます。

### 3. 自動フォーカス設定
- **設定 > 更新時にInspectorをフォーカス**: 
  ONにすると、ローテーション更新が行われた際、そのInspectorウィンドウを自動的に前面（フォーカス状態）にします。

## 🛠️ その他の改善・修正

- **ショートカット追加**: `Ctrl+Shift+L` で全てのInspectorのロック状態を一括切り替えできるようになりました（従来は `Ctrl+L` でアクティブのみ）。
- **完全ローカライズ**: すべてのメッセージとUIテキストを日本語/英語に対応させました。
- **安定性向上**:
  - ドメインリロード（スクリプトコンパイル）後の動作を安定化。
  - メモリリークの要因となっていたイベント購読漏れを修正。
  - サービスロケーターの二重登録エラーを修正。

## ⚠️ 注意事項

- 本バージョンは Unity Editor の内部リフレクションを使用しています。将来のUnityバージョンアップで動作が変わる可能性がありますが、その場合は従来の安全な方式に自動的にフォールバックします。
