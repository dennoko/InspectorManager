# Inspector Manager の使い方

Inspector Managerは、UnityエディタでのInspectorウィンドウの管理を効率化する拡張機能です。
複数のInspectorウィンドウを自動的にローテーションさせて、選択したオブジェクトを順番に表示したり、選択履歴の管理やお気に入り登録が可能です。

## 主な機能

### 🔄 Inspectorローテーション
複数のInspectorウィンドウを開いている場合、通常はすべて同じオブジェクトが表示されてしまいます。
ローテーション機能をONにすると、オブジェクトを選択するたびに、Inspectorウィンドウが順番に更新されます。
これにより、複数のオブジェクトのプロパティを並べて比較したり、親オブジェクトと子オブジェクトを同時に確認したりすることが容易になります。

**フォルダ選択時のブロック機能:**
設定で「フォルダ選択時の更新をブロック」をONにすると、ProjectウィンドウでフォルダをクリックしてもInspectorが更新されません。誤ってInspectorの内容が書き換わるのを防ぎます。

### 📜 選択履歴 (History)
過去に選択したオブジェクトの履歴を自動的に記録します。
履歴タブから過去のオブジェクトを選択し直したり、ブラウザのように「戻る」「進む」操作が可能です。

### ⭐ お気に入り (Favorites)
よく使うオブジェクトをお気に入りに登録できます。
オブジェクトを選択して `Ctrl+D` を押すか、履歴リストからドラッグ＆ドロップで登録できます。

### 🔒 Inspectorオーバーレイ
各Inspectorウィンドウの上部に、現在の番号とロック状態を示すオーバーレイが表示されます。
ボタンをクリックするだけで、そのInspectorのロック状態を切り替えることができます。

## ショートカット一覧

| ショートカット | 機能 |
| --- | --- |
| `Ctrl + L` | 現在アクティブなInspectorのロックを切り替え |
| `Ctrl + Alt + L` | すべてのInspectorのロック状態を一括切り替え |
| `Ctrl + [` | 履歴を戻る |
| `Ctrl + ]` | 履歴を進む |
| `Ctrl + D` | 選択中のオブジェクトをお気に入りに追加/削除 |

## 設定

設定タブでは以下の項目を変更できます。

*   **言語 / Language**: 日本語と英語を切り替えます。
*   **最大履歴数**: 保存する履歴の最大数を指定します。
*   **シーンオブジェクトを記録**: シーン内のオブジェクトを履歴に残すか設定します。
*   **アセットを記録**: Project内のアセットを履歴に残すか設定します。
*   **無効なエントリを自動削除**: 削除されたオブジェクトなどを履歴から自動的に削除します。
*   **フォルダ選択時の更新をブロック**: フォルダ選択時にInspector更新を行わないようにします。

## インストール方法

`Assets/Editor/InspectorManager` フォルダをプロジェクトに配置するだけで使用可能です。
メニューバーの `dennokoworks > Inspector Manager` からウィンドウを開くことができます。
